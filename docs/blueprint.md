# **App Name**: FlowtingAI

## Core Features:

- AI-Powered Chat Interface: A conversational interface powered by AI that supports markdown, tables, and inline media for seamless communication and content generation. The tool presents conversations in a threaded format, creating intuitive communication.
- AI Model Selection: Dropdown menu to select from different AI models, allowing users to switch between models for different tasks and compare their performance and credits available next to it.
- Persona-Based Assistance: Quick persona selectors offer tailored support from AI agents specialized in research, writing, or technical tasks, adapting to diverse user needs. User can add/create custom personas with a dialog box
- Contextual Insight Pinboard: A right sidebar to filter, search and save user-created pins in a structured form to use for later purposes. Users are allowed to take personal notes as well.
- Model Benchmarking Analytics: Tools and features for comparing models, analyzing user activity, and viewing a modelâ€™s latency and pricing. Use these data points to create insightful tables.
- Token Usage Tracker: A token usage progress bar displaying current session usage to effectively manage and control AI costs and resource allocation.
- Theme Switcher: Dark and light theme switcher.
- API Support: Supportable for django and supabase, create production level REST APIs with high security. also, everything should be functional and easy to integrate external APIs and calling

## Style Guidelines:

- Background color: Light gray (#F5F5F5) to create a neutral and professional workspace.
- Primary color: Soft blue (#79A3B1) to evoke trust and intelligence.
- Accent color: Muted purple (#A89BB1) to add a touch of sophistication and modernity.
- Body and headline font: 'Inter', a sans-serif, known for its readability and modern appearance.
- Three-column layout with a left sidebar for navigation, a main chat workspace in the center, and a right sidebar for insight and pinboard functionality.
- Simple, minimalist icons for easy navigation and functionality in a sleek and intuitive AI collaboration platform. Ensure high contrast to enhance visibility.
- Subtle hover effects, transitions, and loading states to create a smooth and premium user experience.
